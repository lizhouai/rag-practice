# RAG 小评测集

这组问题用于跑通最小 RAG 后做人工检查。先看检索 Top-K 有没有命中证据，再看答案有没有忠实引用资料。

| 问题 | 应命中文档 | 理想答案要点 |
| --- | --- | --- |
| 跨境订单退款多久到账？ | `refund_policy.md` | 审核通过后通常 7 到 15 个工作日；超过 15 个工作日需提供订单号、支付流水号、银行卡后四位核查。 |
| 预售商品可以按 48 小时规则催单吗？ | `shipping_policy.md` | 不可以，预售订单以商品详情页承诺的预计发货日期为准。 |
| 会员积分可以提现吗？ | `member_policy.md` | 不能提现，也不能转赠；可兑换优惠券、会员权益和部分活动商品。 |
| 物流 72 小时没更新怎么办？ | `shipping_policy.md` | 可提交物流异常反馈，客服会在 1 个工作日内联系快递公司核查。 |
| 虚拟商品支持无理由退款吗？ | `refund_policy.md` | 不支持无理由退款；质量问题需要人工审核。 |
| 今天北京天气怎么样？ | 无 | 应拒答或说明资料中没有相关信息，不能编造。 |
